## 一、项目定位

面向大学生学习场景的 **AIGC 移动端应用**，基于 Flutter 开发。通过学习日志、课程任务、日历记录、任务进度等数据，结合大模型能力，形成"**记录—执行—分析—复盘**"的学习闭环。

参加 **第三届（2026）中国高校计算机大赛 - AIGC创新赛 应用赛道**。

## 六、AIGC 能力详情

### 1. AI 生成学习日志
- **入口**: AI 学习助手页 → 第一个卡片 / 番茄钟完成弹窗
- **流程**: 输入自然语言 → 生成结构化结果 → 预览编辑 → 一键保存
- **文件**: `lib/src/services/ai_study_service.dart` → `generateStudyLog()`
- **状态**: ✅ DeepSeek 真实 API + Mock 双模式

### 2. AI 任务拆解
- **入口**: AI 学习助手页 → 第二个卡片
- **流程**: 输入复杂任务 → 子任务列表 + 按天安排 → 一键加入（含 subtasks）
- **文件**: `lib/src/services/ai_study_service.dart` → `generateTaskPlan()`
- **状态**: ✅ DeepSeek + Mock 双模式，子任务存独立字段

### 3. AI 分析型周报
- **入口**: AI 学习助手页 → 第三个卡片
- **维度**: 主题 / 投入 / 问题 / 完成率 / 风险 / 评价 / 建议
- **文件**: `lib/src/services/ai_study_service.dart` → `generateWeeklyAnalysis()`
- **状态**: ✅ DeepSeek + Mock 双模式

### 4. AI 风险提醒
- **入口**: AI 学习助手页 → 第四个卡片
- **类型**: 截止风险 / 进度偏低 / 课程断档 / 完成率低 / 记录过少
- **分级**: high (红) / medium (黄) / low (蓝)
- **文件**: `lib/src/services/ai_study_service.dart` → `generateRiskWarnings()`
- **状态**: ✅ DeepSeek + Mock 双模式

### 5. AI 知识闪卡
- **入口**: 侧边菜单 → 知识闪卡
- **流程**: 从学习日志自动生成问答闪卡 → 点击翻转 → 前后翻页
- **文件**: `lib/src/services/ai_study_service.dart` → `generateFlashCards()` + `lib/src/ui/study/flash_card_page.dart`
- **状态**: ✅ DeepSeek + Mock 双模式

### DeepSeek 集成
| 项目 | 内容 |
|------|------|
| 文件 | `lib/src/services/deepseek_client.dart` |
| 模型 | deepseek-v4-flash (默认) / deepseek-v4-pro |
| 认证 | Bearer Auth, API Key 安全存储 (flutter_secure_storage) |
| 输出 | `response_format: json_object`，解析后映射到现有模型 |
| 回退 | 未配置 API Key 时自动使用 mock 演示模式 |
| 设置页 | `lib/src/ui/study/ai_settings_page.dart` (输入/显示/保存/删除/测试) |
| 智能输入 | 图片 OCR (google_mlkit) + 语音输入 (speech_to_text) |

## 十、演示流程 (建议录制)

```
1. 打开 StudyTrace 首页 → 展示学习概览 + 连续打卡
2. 点击「AI 学习助手」 → 进入 AI 页面
3. 输入一句学习情况 → AI 生成结构化日志 → 保存
4. 输入复杂任务 → AI 拆解子任务 → 一键加入任务列表（含子任务）
5. 回到首页查看任务进度
6. 生成 AI 分析周报 → 保存到历史
7. 运行 AI 风险提醒 → 展示风险卡片
8. 侧边菜单 → 知识闪卡 → 翻转浏览
9. 侧边菜单 → 专注计时 → 完成番茄钟 → AI 记录学习
10. 系统设置 → 配置 API Key → 切换模型 → 测试连接
```
